# Public Assets

Place your Humana AI Avatar image here as `humana-avatar.png`.

The image should be:
- High resolution (recommended: 1024x1024 or larger)
- PNG format with transparency (optional)
- Shows the avatar from chest up with the white helmet and Lady Justice emblem

The application will automatically use this image on:
- Landing page (`/`)
- Chat page (`/chat`) with lip sync animation

If the image is not found, a placeholder will be displayed.
